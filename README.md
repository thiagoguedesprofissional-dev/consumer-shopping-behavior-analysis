# Consumer Insights and Shopping Behavior Analysis

## Sobre o Projeto

Este projeto apresenta uma análise detalhada sobre o comportamento de consumo baseada em um conjunto de dados de varejo com 3.900 registros transacionais. O objetivo central foi transformar dados brutos em inteligência de negócio, com foco em métricas de retenção, sazonalidade e otimização de faturamento.

A análise foi estruturada para responder a 18 perguntas estratégicas, abrangendo desde a performance demográfica até o cálculo de Lifetime Value (LTV) e ciclos de recompra estimados.

## Tecnologias e Ferramentas

- **SQL**: Utilizado para extração de dados, limpeza e criação de métricas complexas (CTEs, Case When e Agregações).
- **Power BI**: Desenvolvimento de Dashboard interativo com foco em KPIs executivos e visualização de tendências.
- **LaTeX**: Empregado na redação do relatório técnico e estratégico para garantir rigor formal e clareza documental.
- **Excel/CSV**: Fonte de dados original.

## Principais Insights Extraídos

- **Concentração de Receita**: O público masculino representa mais de 67% do faturamento total, indicando uma oportunidade clara de expansão de market share no nicho feminino.
- **Saúde da Base (LTV)**: Clientes categorizados como fiéis (mais de 30 compras anteriores) detêm a maior fatia da receita acumulada, validando estratégias de upselling para este segmento.
- **Logística como Alavanca**: Identificou-se uma correlação positiva entre métodos de envio expressos e tickets médios superiores, sugerindo o uso do frete rápido como benefício para conversão de alto valor.
- **Dinâmica Sazonal**: Picos de vendas distintos por gênero (Outono para mulheres e Inverno para homens) permitem a criação de calendários de marketing e gestão de estoque mais assertivos.
